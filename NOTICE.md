## Access control & the Sensory Profile
This is **not an open public deployment**. The entire application is access-gated: every instrument
ships **encrypted** and is readable only by a small set of individually keyed, authorized users
(non-commercial, personal/educational use). The plaintext instruments are **not** present in this
public repository — only ciphertext is.

`sensorial` (Adolescent/Adult Sensory Profile) is a **commercial Pearson product**, marked
`copyrighted: true` in its data module. It is included only behind the access gate for authorized
private use. If you are a rights-holder and want it removed, contact the repository owner and it will
be taken down promptly.

## PII removed
The source PDFs were blank-form templates except the Sensory Profile, which arrived with a real
person's identifying data and answers filled in. None of that was carried over — all forms here
are blank and collect no name, date of birth, or clinician fields.
